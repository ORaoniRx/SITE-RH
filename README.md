# RH-flow
Um Site que ajuda na organização das demandas das empresas e na entrega de formularios,documentos etc
Gustavo Henrique , Raoni Ramos , Joao Rafael e Izadora caceres
Database
--------
feita no PGadmin4
To initialize the PostgreSQL schema and seed example data:

```bash
cd backend
node db/seed.js
```

The server runs `initDb()` on startup which applies `backend/db/schema.sql`.
